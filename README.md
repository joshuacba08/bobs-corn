# Bob's corn

Bob's corn es un proyecto fullstack que de manera simple y minimalista permite el granjero Bob venda una unidad de maiz por minuto por cada `id` de cliente.

## Requerimientos

- Node.js 22.0.0
- pnpm 6.0.0 o superior
- PostgreSQL
- Docker
